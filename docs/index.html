PeerFold

One task: review a scientific paper. Opens in a native window — no browser tab juggling. Select text, type your comment, move on. Citations open in a popup. Highlights save as standard PDF annotations you can share with authors.

Quick start

$ pipx install peerfold-review
$ peerfold manuscript.pdf --reviewer RB

Opens an embedded window (pip/pipx). Standalone binaries use your system browser.

Install

  • pipx (recommended)

    Isolated CLI on your PATH. Requires Python 3.10+.

    $ pipx install peerfold-review
  • pip

    Inside an existing virtual environment.

    $ pip install peerfold-review
  • macOS

    Recommended: peerfold-macos.dmg — open it, drag PeerFold.app to Applications, double-click and pick your PDF.

    Or download peerfold-macos.command — double-click to open Terminal (not Safari).

  • Linux

    peerfold-linux — single self-extracting file.

    $ chmod +x peerfold-linux
    $ ./peerfold-linux manuscript.pdf --reviewer RB
  • Windows

    peerfold-win.exe — single self-extracting installer-style binary.

    > peerfold-win.exe manuscript.pdf --reviewer RB

Built for peer review

  • Click and type — select a span, comment immediately, no dialog maze
  • Comments pane stays beside the PDF; jump between notes in one click
  • Numbered citations open DOIs and URLs in a popup window
  • Autosave, undo, multi-tab sync for long review sessions
  • Exports paper_RB-2026-06-09.pdf with Acrobat-compatible highlights

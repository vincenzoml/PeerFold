PeerFold

One task: review a scientific paper. Opens in a native window — no browser tab juggling. Select text, type your comment, move on. Citations open in a popup. Highlights save as standard PDF annotations you can share with authors.

Install

macOS & Linux

Detects your OS, downloads the latest release, and installs. Opens in a native window (no browser URL bar). Use peerfold … --browser for the system browser instead.

$ curl -fsSL https://vincenzoml.github.io/PeerFold/install.sh | bash

Windows

PowerShell — installs to %LOCALAPPDATA%\Programs\PeerFold and adds it to your user PATH.

> irm https://vincenzoml.github.io/PeerFold/install.ps1 | iex

Python

If you already use Python 3.10+. Same native window as the standalone app.

$ pipx install peerfold-review
$ peerfold manuscript.pdf --reviewer RB

Built for peer review

  • Click and type — select a span, comment immediately, no dialog maze
  • Comments pane stays beside the PDF; jump between notes in one click
  • Numbered citations open DOIs and URLs in a popup window
  • Autosave, undo, multi-tab sync for long review sessions
  • Exports paper_RB-2026-06-09.pdf with Acrobat-compatible highlights

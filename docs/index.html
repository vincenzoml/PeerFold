PeerFold

Review a PDF in a native window. Select text, type a comment, move on. Highlights save as standard annotations — share the exported file with authors.

Install app

macOS & Linux — native window, no Python needed.

$ curl -fsSL https://vincenzoml.github.io/PeerFold/install.sh | bash

Windows

> irm https://vincenzoml.github.io/PeerFold/install.ps1 | iex

Over SSH: peerfold paper.pdf --web

Install with pip

Python 3.10+

$ python3 -m pip install --user peerfold-review $ peerfold manuscript.pdf --reviewer RB

Shared folder or git repo

Add one launcher script. It creates .venv-peerfold/ locally, installs a pinned PyPI version, and runs PeerFold — so co-authors stay on the same build. Commit scripts/peerfold.py; gitignore the venv.

One-liner (from your project root):

$ mkdir -p scripts && curl -fsSL https://vincenzoml.github.io/PeerFold/peerfold.py -o scripts/peerfold.py && echo '.venv-peerfold/' >> .gitignore

Or download

peerfold.py → save as scripts/peerfold.py in your project. Add .venv-peerfold/ to .gitignore.

Run

$ python3 scripts/peerfold.py manuscript.pdf --reviewer RB

No PDF path opens the welcome screen (drag-drop or Open). Over SSH: append --web.

Upgrade (when a new release is out):

$ python3 scripts/peerfold.py --update

Then commit scripts/peerfold.py so everyone pulls the new pin.

Features

  • Span highlights + comment threads beside the PDF
  • Citations open in a popup; autosave and undo
  • Exports paper_RB-2026-06-09.pdf with Acrobat-compatible marks

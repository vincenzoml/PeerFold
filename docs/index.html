PaperTrail

Review PDFs in your browser. Highlights are real PDF annotations — readable in Acrobat, Preview, and every reader.

$ pipx install papertrail-review
$ papertrail manuscript.pdf --reviewer VC

Features

  • Span-accurate highlights with threaded comments
  • Citation links open DOIs and URLs directly
  • Autosave, multi-tab sync, reviewer handoff
  • Saved as paper_VC-2026-06-09.pdf beside the original

Install

$ pip install papertrail-review
or download a standalone binary from Releases

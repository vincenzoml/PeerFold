PeerFold

PDF annotation for paper reviews. Click, type, done.

Notes become standard PDF highlights — open in any reader, attach to an email, no export step.

Drop one launcher in your paper source folder; the pinned version rides along in git so co-authors stay aligned.

Install

macOS / Linux

Windows

Python 3.10+ · peerfold paper.pdf --reviewer AB · SSH: add --web

Paper directory

Add the launcher once; commit it, gitignore .venv-peerfold/. Everyone runs the same pinned build from the shared folder.

python3 scripts/peerfold.py paper.pdf · new release: --update, then commit the script.

PeerFold

Highlight and comment on a PDF like you would on paper — in a small native app, not a browser tab.

Comments are written straight into the PDF as normal highlights. Send the file back; authors open it in Acrobat, Preview, or whatever they already use. No proprietary format, no extra export step.

Install

macOS / Linux

Windows

Python 3.10+ — peerfold paper.pdf --reviewer AB. Over SSH add --web.

Shared project folder

Drop one script in the repo so everyone runs the same version. It keeps a local venv; commit the script, gitignore .venv-peerfold/.

Then python3 scripts/peerfold.py paper.pdf · upgrade with --update and commit the script.
